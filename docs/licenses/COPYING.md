# Multilicense project

SNES-IDE is a multi-license software, this means that there are different source codes which requires different licenses for copying. All licenses of SNES-IDE are listed in this directory:

- Apache v2
- BSD
- GPL v2
- GPL v3
- MIT
- Xiph

GPLv3 is the main license because the SNES-IDE source code is licensed with GNU GPL v3, the other licenses are from libraries, binaries, tools and other useful technologies for snes game development. Using and copying them, you must respect their respective terms, which are found in their directories as COPYING or LICENSE. Some of those projects are also multilicense, and some parts of them are licensed with one license and another part with another license. Any question about the project's license open an issue in [github](https://github.com/BrunoRNS/SNES-IDE/issues).
